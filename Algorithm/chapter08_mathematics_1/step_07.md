# step 7
- [2869 번](https://www.acmicpc.net/problem/2869)
## 난이도
- 브론즈 1
## 핵심
- 입력받은 수 범위 탐색

## 풀이
```c++
#include <iostream>
#include <string>

int main(){
    std::ios::sync_with_stdio(false);
    std::cin.tie(nullptr);
    
    long long a, b, v;
    std::cin >> a >> b >> v;

    std::cout << ((v-b-1)/(a-b)) + 1 << '\n';

    return 0;
}

// C++의 기본 나눗셈은 나머지를 '버림'
// 하지만 현재 필요한 것은 올림 나눗셈.
// 따라서 분자에 분모-1을 더해줌(v-a + (a-b-1)) = v-b-1
// 계산식의 핵심 개념은 도착하는 날의 전날에 미끄러진 위치는 올라갈 수 있는 거리보다 작음.
// 따라서 마지막 날 도착하는 +1을 마지막에 더해줘야함.
// 첫날 ~ 마지막 전날 까지의 소요일은 올라가고 미끄러진 총 결과값으로 마지막 날 올라갈 수 있는 거리를 제외한 거리를 충족해야함.
// 이 계산이 (하루에 올라가는 거리 - 하루에 미끄러질 거리) / (총 거리 - 하루에 올라가는 거리)
// 이렇게 마지막 날 올라가며 도착하는 거리를 제외한 마지막 날 전까지 올라가야 하는 소요일을 구할 수 있음.
```
