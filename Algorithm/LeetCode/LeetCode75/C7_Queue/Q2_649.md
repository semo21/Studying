# 649. Dota2 Senate

## Difficulty
- Medium

## Point
- Store each senator's index in a queue based on their party.
- Compare the front indices of both queues to determine which senator acts first.
- The earlier senator bans the later senator.
- The surviving senator is pushed back with index + `n` to represent the next round.

## Code
```c++
class Solution {
public:
    string predictPartyVictory(string senate) {
        int n = senate.size();

        queue<int> rIdx;
        queue<int> dIdx;

        for(int i = 0; i < n; ++i){
            if(senate[i] == 'R'){
                rIdx.push(i);
            }else{
                dIdx.push(i);
            }
        }

        while(!rIdx.empty() && !dIdx.empty()){
            int r = rIdx.front();
            int d = dIdx.front();

            if(r < d){
                rIdx.push(r + n);
                
            }else if(r > d){
                dIdx.push(d + n);
            }

            rIdx.pop();
            dIdx.pop();
        }

        return rIdx.empty() ? "Dire" : "Radiant";
    }
};
```

### Constraints
- n == senate.length
- 1 <= n <= 104
- senate[i] is either 'R' or 'D'.

## Explanation
- Store the indices of Radiant senators in `rIdx` and Dire senators in `dIdx`.
- In each round, compare the front index of both queues.
- The senator with the smaller index acts first and bans the other senator.
- The banned senator is removed from the queue.
- The surviving senator is pushed back with `index + n`, which means they will act again in the next cycle.
- When one queue becomes empty, the other party wins.

## Complexity
### Time Complexity: O(n)
- The input string is traversed once to initialize the queue.
- During the simulation, each iteration removes one senator from futue participation.
- Queue operations are constant time.
- Therefore, the total time complexity is O(n).

### Space Complexity: O(n)
- Two queues store senator indices.
- The total number of stored indices is at most proportional to `n`.
