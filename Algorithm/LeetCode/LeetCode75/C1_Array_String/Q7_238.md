# 238. Product of Array Except Self

## Difficulty
- Medium

## Point
- 

## Code
```c++
class Solution {
public:
    vector<int> productExceptSelf(vector<int>& nums) {
        int n = nums.size();

        vector<int> answer(n);

        answer[0] = 1;
        for (int i = 1; i < n; ++i) {
            answer[i] = answer[i - 1] * nums[i - 1];
        }

        int suffix = 1;
        for (int i = n - 1; i >= 0; --i) {
            answer[i] *= suffix;
            suffix *= nums[i];
        }

        return answer;
    }
};
```

### Constraints:
- 2 <= nums.length <= 105
- -30 <= nums[i] <= 30
- The input is generated such that answer[i] is guaranteed to fit in a 32-bit integer.
 
## Explanation
- 나눗셈 금지
- 따라서 i 왼쪽의 곱들을 저장한 prefix배열과 오른쪽의 곱들을 저장한 suffix 배열을 만들어 answer[i] = prefix[i] * suffix[i]로 answer를 생성
- 본문의 코드는 해당 과정을 더욱 최적화한 코드
- prefix를 answer에 우선 저장, 이후 suffix의 값을 따라 반복문을 순회하며 answer의 값에 suffix를 곱해주며 answer[i]를 차례대로 수정
- prefix와 suffix배열을 생성하지 않고도 같은 방법으로 풀이 가능
