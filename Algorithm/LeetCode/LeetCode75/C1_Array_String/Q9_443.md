# 443. String Compression

## Difficulty
- Medium

## Point
- if 문의 조건 설정, chars[idx++] 할당

## Code
```c++
// 초기 코드
class Solution {
public:
    int compress(vector<char>& chars) {
        char c = chars[0];
        int count = 1;
        int idx = 0;

        for(int i = 1; i <= chars.size(); ++i){
            if(i == chars.size() || c != chars[i]){
                chars[idx++] = c;
                if(count > 1){
                    for(char i : to_string(count)){
                        chars[idx++] = i;
                    }
                }
                if(i < chars.size()){
                    c = chars[i];
                    count = 1;
                }
                
            }else{
                count++;
            }
        }

        return idx;
            
    }
};

// 최적화 후
class Solution {
public:
    int compress(vector<char>& chars) {
        int n = chars.size();

        int write = 0;
        int i = 0;
        
        while(i < n){
            char current = chars[i];

            int start = i;
            while(i < n && chars[i] == current){
                i++;
            }

            int count = i - start;

            chars[write++] = current;

            if(count > 1){
                for(char digit : to_string(count)){
                    chars[write++] = digit;
                }
            }
        }
        

        return write;
    }
};
```

### Constraints:
- 1 <= chars.length <= 2000
- chars[i] is a lowercase English letter, uppercase English letter, digit, or symbol.
 
## Explanation
- 판단 시점은 1. 문자가 변할 때, 2. 문자열이 끝날 때
- 판단 시점에서 덮어쓰기를 시작
- 덮어쓸 위치를 idx로 저장하며 기억
