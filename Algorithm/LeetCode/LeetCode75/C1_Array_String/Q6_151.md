# 151. Reverse Words in a String

## Difficulty
- Medium

## Point
- 

## Code
```c++
class Solution {
public:
    string reverseWords(string s) {
        vector<string> words;
        string word;
        for(int i = 0; i < s.size(); ++i){
            if(s[i] != ' '){
                word += s[i];
            }else{
                if(!word.empty()){
                    words.push_back(word);
                    word.clear();
                }
            }
        }

        if(!word.empty()){
            words.push_back(word);
        }

        string result;
        for(int i = words.size()-1 ; i >= 0; --i){
            result += words[i];
            if(i > 0){
                result += ' ';
            }
        }

        return result;
    }
};
```

### Constraints:
- 1 <= s.length <= 104
- s contains English letters (upper-case and lower-case), digits, and spaces ' '.
- There is at least one word in s.
 
## Explanation
- 속도 0ms, 메모리 11.22mb 로 메모리최적화가 덜 된 풀이
- empty와 clear를 활용하여 공백과 단어를 구분하며 vector에 저장
- 저장한 단어들을 거꾸로 꺼내며 result에 더한 후 반환
