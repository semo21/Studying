# 2390. Removing Stars From a String

## Difficulty
- Medium

## Point
- Use `string` as a `stack`

## Code
```c++
class Solution {
public:
    string removeStars(string s) {
        string result;

        for (char c : s){
            if(c != '*'){
                result.push_back(c);
            }else{
                result.pop_back();
            }
        }

        return result;
    }
};
```

### Constraints:
- 1 <= s.length <= 105
- s consists of lowercase English letters and stars *.
- The operation above can be performed on s.

## Explanation
- Use `string` as a `stack`
- If the current character is `*`, remove the last character using `pop_back()`
- Otherwise, append the cahraacter using `push_back()`

## Complexity
### Time Complexity: O(n)
- The string is traversed once

### Space Complextity: O(n)
- The result string may store up to n characters
