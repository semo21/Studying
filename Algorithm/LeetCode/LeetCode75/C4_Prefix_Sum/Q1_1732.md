# 1732. Find the Highest Altitude

## Difficulty
- Easy

## Point
- gain[i]를 더해가며 highest를 매 순간 갱신

## Code
```c++
class Solution {
public:
    int largestAltitude(vector<int>& gain) {
        int highestAltitude = 0;
        int currentAltitude = 0;
        
        for(int a : gain){
            currentAltitude += a;
            highestAltitude = max(currentAltitude, highestAltitude);    
        }

        return highestAltitude;
    }
};
```

### Constraints:
- n == gain.length
- 1 <= n <= 100
- -100 <= gain[i] <= 100
 
## Explanation
- currentAltitude에 현재까지의 누적고도 저장
- gain을 순회하며 current를 갱신하며 highest와 비교하며 최고 고도를 갱신
- 시작 고도인 0 또한 최고 고도가 될 수 잇으므로 highest또한 0으로 초기화

## Complexity
### Time Complexity: O(n)
- 하나의 배열을 한 번만 순회
### Space Complextity: O(1)
- 추가 변수만 사용
