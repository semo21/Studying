# 724. Find Pivot Index

## Difficulty
- Easy

## Point
- total = rightSum + leftSum + pivotIndex
- 위 등식이 성립하지 않는다면 pivot은 존재하지 않음

## Code
```c++
class Solution {
public:
    int pivotIndex(vector<int>& nums) {
        int total = 0;
        for(int num : nums){
            total += num;
        }

        int leftSum = 0;
        for(int i = 0; i < nums.size(); ++i){
            int rightSum = total - leftSum - nums[i];

            if(leftSum == rightSum){
                return i;
            }
            leftSum += nums[i];
        }

        return -1;
    }
};
```

### Constraints:
- 1 <= nums.length <= 104
- -1000 <= nums[i] <= 1000
 
## Explanation
- total을 먼저 구해둠
- 이후 rightSum = total - leftSum - nums[i]
- rightSum == leftSum 이라면 pivot은 i, return i
- 순회가 끝나도록 없다면 pivot은 없으므로 -1 반환

## Complexity
### Time Complexity: O(n)
- 하나의 배열을 한 번만 순회
### Space Complextity: O(1)
- 추가 변수만 사용
