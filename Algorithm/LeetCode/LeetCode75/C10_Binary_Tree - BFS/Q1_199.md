# 199. Binary Tree Right Side View

## Difficulty
- Medium

## Point
- Use BFS level-order traversal.
- Store the last node of each level.
- Use `levelSize` to precess only the nodes in the current level.

## Code
```c++
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
 *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left), right(right) {}
 * };
 */
class Solution {
public:
    vector<int> rightSideView(TreeNode* root) {
        vector<int> result;
        queue<TreeNode*> q;

        if(root == nullptr) return result;
        q.push(root);

        while(!q.empty()){
            int levelSize = q.size();
            
            for(int i = 0; i < levelSize; ++i){
                TreeNode* cur = q.front();
                q.pop();
                if(i == levelSize-1){
                    result.push_back(cur->val);                
                }
                
                if(cur->left != nullptr)    q.push(cur->left);
                if(cur->right != nullptr)   q.push(cur->right);                
            }
        }

        return result;
    }
};
```

### Constraints
- The number of nodes in the tree is in the range [0, 100].
- -100 <= Node.val <= 100

## Explanation
- If `root` is `nullptr`, return `result`.
- Push `root` into the queue.
- Traveerse the tree while queue is not empty.
- At the start of each level, store `q.size()` as `levelSize`.
- Process exactly `levelSize` nodes, so newly pushed child nodes are handled in the next level.
- If `i == levelSize - 1`, the current node is the last node of that level, so add `cur->val` to `result`.
- Push `cur->left` and `cur->right` into the queue if they are not `nullptr`.
- Return `result`.

## Complexity
### Time Complexity: O(n)
- Each node is pushed into and popped from the queue once.

### Space Complexity: O(n)
- The queue can store up to O(n) nodes in the worst case.
- The `result` vector can also store up to O(n) values.
