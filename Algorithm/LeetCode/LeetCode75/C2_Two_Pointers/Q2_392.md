# 392. Is Subsequence

## Difficulty
- Easy

## Point
- 빈 문자열 처리, s와 idx길이 비교

## Code
```c++
class Solution {
public:
    bool isSubsequence(string s, string t) {
        if(s.empty())   return true;

        int n = t.size();
        int idx = 0;
        for(int i = 0; i < n; ++i){
            if(t[i] == s[idx]){
                idx++;
                if(idx == s.size()){
                    return true;
                }
            }
        }

        return false;
    }
};
```

### Constraints:
- 0 <= s.length <= 100
- 0 <= t.length <= 104
- s and t consist only of lowercase English letters.
 
## Explanation
- s가 빈 문자열일 경우, t의 모든 경우의 Subsequence가 됨 -> return true
- 이후 한 글자씩 s의 요소와 비교하며 idx를 증가하며 idx가 s의 길이와 같아지면 true반환
- idx가 s보다 짧다면 false
