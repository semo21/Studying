# 11. Container With Most Water

## Difficulty
- Medium

## Point
- maxArea와 area를 계속해서 비교하며 저장
- left와 right 중 높이가 더 작은쪽을 계속해서 이동시키기

## Code
```c++
class Solution {
public:
    int maxArea(vector<int>& height) {        
        int left = 0;
        int right = height.size() - 1;
        int maxArea = 0;

        while(left < right){
            int area = (right-left) * (height[left] > height[right] ? height[right] : height[left]);
            maxArea = max(maxArea, area);
            if(height[left] < height[right]){
                left++;
            }else{
                right--;
            }
        }

        return maxArea;
    }
};
```

### Constraints:
- n == height.length
- 2 <= n <= 105
- 0 <= height[i] <= 104
 
## Explanation
- maxArea를 선언하여 최댓값을 계속 저장
- left 혹은 right를 이동하며 매번 넓이 계산
- left와 right중 height가 낮은 부분을 계속해서 이동시키기
