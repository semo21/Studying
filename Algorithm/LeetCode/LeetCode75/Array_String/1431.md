# 1431. Kids With the Greatest Number of Candies

## Difficulty
- Easy

## Point
- 

## Code
```c++
class Solution {
public:
    vector<bool> kidsWithCandies(vector<int>& candies, int extraCandies) {
        int max = 0;
        for(int c : candies){
            if(c > max){
                max = c;
            }
        }

        vector<bool> result;

        for(int c : candies){
            result.push_back(c + extraCandies >= max);
        }

        return result;        
    }
};
```

### Constraints:
- n == candies.length
- 2 <= n <= 100
- 1 <= candies[i] <= 100
- 1 <= extraCandies <= 50
## Explanation
- extraCandies를 받고서도 최댓값이 못되는 경우는 candies[i] + extraCandies < max(candies)의 경우 뿐임
- candies의 최댓값을 구한 뒤, 다시 한 번 모든 요소를 순회하며 result에 해당 판별 값을 push_back함
