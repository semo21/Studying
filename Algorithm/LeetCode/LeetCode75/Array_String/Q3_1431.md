# 1431. Kids With the Greatest Number of Candies

## Difficulty
- Easy

## Point
- 

## Code
```c++
class Solution {
public:
    vector<bool> kidsWithCandies(vector<int>& candies, int extraCandies) {
        int max = 0;
        for(int c : candies){
            if(c > max){
                max = c;
            }
        }

        vector<bool> result;

        for(int c : candies){
            result.push_back(c + extraCandies >= max);
        }

        return result;        
    }
};
```

### Constraints:
- n == candies.length
- 2 <= n <= 100
- 1 <= candies[i] <= 100
- 1 <= extraCandies <= 50
  
## Explanation
- extraCandies를 받은 뒤에도 최대 사탕 개수보다 작다면, 해당 아이는 최댓값을 가질 수 없음
