# 643. Maximum Average Subarray I

## Difficulty
- Easy

## Point
- subarray를 추적하며 maxSum 갱신

## Code
```c++
class Solution {
public:
    double findMaxAverage(vector<int>& nums, int k) {
        int windowSum = 0;
        for(int i = 0; i < k; ++i){
            windowSum += nums[i];
        }

        int maxSum = windowSum;
        for(int i = k; i < nums.size(); ++i){
            windowSum += nums[i] - nums[i-k];
            maxSum = max(maxSum, windowSum);
        }        

        return static_cast<double>(maxSum) / k;

    }
};
```

### Constraints:
- n == nums.length
- 1 <= k <= n <= 105
- -104 <= nums[i] <= 104
 
## Explanation
- 길이가 k인 subarray(window)를 추적.
- 초기 maxSum = windowSum으로 설정하고 subarray가 이동할 때마다 변화를 max()함수로 갱신시도
- 반환값은 double로 형변환
