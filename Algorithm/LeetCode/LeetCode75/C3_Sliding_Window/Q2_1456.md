# 1456. Maximum Number of Vowels in a Substring of Given Length

## Difficulty
- Medium

## Point
- isVowel 함수를 따로 생성
- subarray를 추적하며 slideVowels 갱신

## Code
```c++
class Solution {
public:
    bool isVowel(char c){
        return 
        c == 'a'||
        c == 'e'||
        c == 'i'||
        c == 'o'||
        c == 'u';
    }

    int maxVowels(string s, int k) {
        int slideVowels = 0;
        for(int i = 0; i < k; ++i){
            if(isVowel(s[i])){
                slideVowels++;
            }
        }

        int maxVowels = slideVowels;
        for(int i = k; i < s.size(); ++i){
            if(isVowel(s[i]))   slideVowels++;
            if(isVowel(s[i-k])) slideVowels--;

            if(slideVowels > maxVowels) maxVowels = slideVowels;
        }

        return maxVowels;
    }
};
```

### Constraints:
- 1 <= s.length <= 105
- s consists of lowercase English letters.
- 1 <= k <= s.length
 
## Explanation
- isVowel함수를 생성하여 모음 판단
- 초기 slideVowels와 maxVowels 설정 후 slideVowels를 순회하며 갱신
- maxVowels와 slideVowels 비교하며 maxVowels 갱신

## Complexity
### Time Complexity: O(n)
- 배열을 한 번만 순회
### Space Complextity: O(1)
- 변수들만 사용
