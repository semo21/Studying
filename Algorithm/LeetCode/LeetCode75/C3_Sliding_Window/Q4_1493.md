# 1493. Longest Subarray of 1's After Deleting One Element

## Difficulty
- Medium

## Point
- 투 포인터와 zeroCount를 활용
- zeroCount가 1보다 클 때, zeroCount를 줄여가며 left를 이동

## Code
```c++
class Solution {
public:
    int longestSubarray(vector<int>& nums) {
        int zeroCount = 0;
        int left = 0;
        int answer = 0;

        for(int right = 0; right < nums.size(); ++right){
            if(nums[right] == 0){
                zeroCount++;
            }

            while(zeroCount > 1){
                if(nums[left] == 0){
                    zeroCount--;
                }
                left++;
            }
            
            answer = max(answer, right - left);
        }

        return answer;
    }
};
```

### Constraints:
- 1 <= nums.length <= 105
- nums[i] is either 0 or 1.
 
## Explanation
- right를 따라 순회하며 zeroCount 누적
- zeroCount가 1개보다 많아진다면, left를 당겨오며 zeroCount--
- 총 길이는 right - left가 됨(마지막에 1개를 반드시 빼야하므로 -1)

## Complexity
### Time Complexity: O(n)
- for(right), while(left)를 해도 최대 O(2n)이므로 O(n)
### Space Complextity: O(1)
- 배열이 아닌 단순 변수들만 사용
