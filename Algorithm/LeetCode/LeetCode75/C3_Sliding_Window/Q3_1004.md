# 1004. Max Consecutive Ones III

## Difficulty
- Medium

## Point
- 투포인터를 활용하여 zeroCount를 세어가며 진행
- zeroCount를 유동적으로 변화시키며 zeroCount를 포함한 Consecutive 1의 길이를 구하기

## Code
```c++
class Solution {
public:
    int longestOnes(vector<int>& nums, int k) {
        int left = 0;
        int zeroCount = 0;
        int answer = 0;

        for(int right = 0; right < nums.size(); ++right){
            if(nums[right] == 0){
                zeroCount++;
            }

            while(zeroCount > k){
                if(nums[left] == 0){
                    zeroCount--;
                }
                left++;
            }
            answer = max(answer, right - left + 1);
        }

        return answer;

    }
};
```

### Constraints:
- 1 <= nums.length <= 105
- nums[i] is either 0 or 1.
- 0 <= k <= nums.length
 
## Explanation
- zeroCount를 세어가며 배열을 순회
- zeroCount가 다 찬 순간부터 왼쪽의 zeroCount를 줄이고 다음 right를 확인하며 
- 0을 포함한 연속되는 1의 길이를 구함
- right를 움직일 때마다 max갱신

## Complexity
### Time Complexity: O(n)
- for(right), while(left)를 해도 최대 O(2n)이므로 O(n)
### Space Complextity: O(1)
- 배열이 아닌 단순 변수들만 사용
