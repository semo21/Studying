# 1207. Unique Number of Occurrences

## Difficulty
- Easy

## Point
- map의 key-value를 이용한 숫자별 occurence 기록
- 이후 set를 만들어 occurence가 set에 이미 존재한다면 false, set에 존재하지 않다면 insert

## Code
```c++
class Solution {
public:
    bool uniqueOccurrences(vector<int>& arr) {
        unordered_map<int, int> freq;

        for(int num : arr){
            freq[num]++;
        }

        unordered_set<int> occur;
        for(auto& pair : freq){
            if(occur.count(pair.second)){
                return false;
            }else{
                occur.insert(pair.second);
            }
        }

        return true;
    }
};
```

### Constraints:
- 1 <= arr.length <= 1000
- -1000 <= arr[i] <= 1000
 
## Explanation
- int타입 키와 값을 가지는 map을 생성
- arr을 순회하며 어떤 숫자의 등장횟수를 기록
- set를 생성하여 occurence만 확인
- 이미 있다면 false반환, 없다면 insert하여 새롭게 기록

## Complexity
### Time Complexity: O(n)
- 최악의 경우 O(2n)까지 늘어남, 따라서 O(n)
### Space Complextity: O(n)
- map과 set를 사용. 
- 최악의 경우 O(2n), 따라서 O(n)
