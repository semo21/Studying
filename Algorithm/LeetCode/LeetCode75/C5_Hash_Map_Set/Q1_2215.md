# 2215. Find the Difference of Two Arrays

## Difficulty
- Easy

## Point
- set를 활용하여 중복되는 요소 축약

## Code
```c++
class Solution {
public:
    vector<vector<int>> findDifference(vector<int>& nums1, vector<int>& nums2) {
        unordered_set<int> set1(nums1.begin(), nums1.end());
        unordered_set<int> set2(nums2.begin(), nums2.end());

        vector<int> onlyNums1;
        vector<int> onlyNums2;

        for(int num : set1){
            if(set2.find(num) == set2.end()){
                onlyNums1.push_back(num);
            }
        }

        for(int num : set2){
            if(set1.find(num) == set1.end()){
                onlyNums2.push_back(num);
            }
        }

        return {onlyNums1, onlyNums2};
    }
};
```

### Constraints:
- 1 <= nums1.length, nums2.length <= 1000
- -1000 <= nums1[i], nums2[i] <= 1000
 
## Explanation
- nums1과 nums2를 unordered_set로 변환, 이 과정에서 중복되는 요소 제거
- 각 set를 순회하며 중복체크, 중복되지 않는다면 각 onlyNums 벡터에 추가 

## Complexity
### Time Complexity: O(n + m)
- n = nums1.size(), m = nums2.size()
- set1과 set2를 1번씩만 순회
### Space Complextity: O(n + m)
- 두개의 hash set와 두개의 결과값 vector 사용됨
- set과 vector의 각 가능한 최대길이는 nums1.size()와 nums2.size()
- 따라서 set1, set2 = n + m / onlyNums1, onlyNums2 = n + m
- O(2n + 2m) = O(n + m)
