# 437. Path Sum III

## Difficulty
- Medium

## Point
- Treat every node as a possible starting point.
- Use `countFrom` to count downward paths from one fixed starting node.
- Use `long long` for the remaining target to avoid overflow.

## Code
```c++
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
 *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left), right(right) {}
 * };
 */
class Solution {
public:
    int pathSum(TreeNode* root, int targetSum) {
        if(root == nullptr) return 0;
        int path = 0;

        path += countFrom(root, targetSum);        
        path += pathSum(root->left, targetSum);
        path += pathSum(root->right, targetSum);
        
        return path;
    }

    int countFrom(TreeNode* node, long long target){
        if(node == nullptr) return 0;
        
        int count = 0;

        if(node->val == target){
            count++;
        }
        
        count += countFrom(node->left, target - node->val);
        count += countFrom(node->right, target - node->val);

        return count;
    }
};
```

### Constraints
- The number of nodes in the tree is in the range [0, 1000].
- -109 <= Node.val <= 109
- -1000 <= targetSum <= 1000

## Explanation
- If `root` is `nullptr`, return 0.
- `pathSum` checks every node as a possible starting point.
- `countFrom` counts paths that start from the current node and go downward.
- In `countFrom`, subtract the current node's value from `target` while moving to child nodes.
- If `node->val == target`, one valid path is found.
- `long long` is used for `target` to prevent integer overflow during subtraction.

## Complexity
### Time Complexity: O(n^2)
- `pathSum` visits every node as a possible starting point.
- For each starting node, `countFrom` may traverse its descendants.
- In the worst case, such as a skewed tree, this becomes O(n^2).
