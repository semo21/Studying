# 1372. Longest ZigZag Path in a Binary Tree

## Difficulty
- Medium

## Point
- Use DFS with direction and current length.
- If the next direction is different, increase the length.
- If the next direction is same, restart the ZigZag length from 1.
- Keep updating the maximum length.

## Code
```c++
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
 *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left), right(right) {}
 * };
 */
class Solution {
public:
    int longestZigZag(TreeNode* root) {
       return max(dfs(root, false, 0), dfs(root, true, 0));       
    }

    int dfs(TreeNode* node, bool isLeft, int length){
        if(node == nullptr) return length-1;
        
        if(isLeft){
            length = max(dfs(node->right, false, length+1), dfs(node->left, true, 1));            
        }else{
            length = max(dfs(node->left, true, length+1), dfs(node->right, false, 1));
        }

        return length;
    }
};
```

### Constraints
- The number of nodes in the tree is in the range [1, 5 * 104].
- 1 <= Node.val <= 100

## Explanation
- `dfs(node, isLeft, legnth)` traverses while carrying the previous direction and current ZigZag length.
- If the next move changes direction, increase `length` by 1.
- If the next move goes in the same direction, restart the length from 1.
- When `node` is `nullptr`, return `length - 1` because the recursion has moved one step beyond a valid node.
- The answer is the maximum value returned from both possible starting directions.

## Complexity
### Time Complexity: O(n)
- Each node can be visited with two direction states.
- This is O(2n), which simplifies to O(n).

### Space Complexity: O(n)
- Recursive calls use the call stack.
- In the worst case, the tree can be skewed, so the recursion depth can be n.
