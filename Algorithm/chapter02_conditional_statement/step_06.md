# step 6
- [2525번](https://www.acmicpc.net/problem/2525)
## 난이도
- 브론즈 3
## 핵심
- 시간 계산 (24시간, 분 단위 순환)

## 풀이
```c++
#include <iostream>

int main(){
    int a, b, c;
    std::cin >> a >> b;
    std::cin >> c;
    
    int h = c/60;
    int m = c%60;

    if(b+m >= 60){
        a+=h+1;
        b+=m-60;        
    }else{
        a+=h;
        b+=m;        
    }

    if(a>=24){
        a-=24;
    }
    std::cout << a << " " << b << std::endl;

    return 0;
}

// 0 <= a <= 23, 0 <= b <= 59 
// 0 <= c <= 1000
// 분 덧셈이 60 이상이라면:
// 시간 +1 추가 계산
// 시간 덧셈이 24 이상이라면:
// 시간 -24 뺄셈 추가실행 (다음 날로 취급하지만, 문제에선 날짜를 출력하지 않음.)
// 시간 24 이상 조건문은 중복을 방지하기 위해 바깥으로 빼서 한 번만 실행

// b += c;
// a += b/60;
// b %= 60;
// a %= 24;
// 위 4줄의 연산으로도 처리 가능
```
